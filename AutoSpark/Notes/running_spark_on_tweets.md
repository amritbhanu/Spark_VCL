# How to Run basic Spark Job on tweets.json dataset

1. The tweets.json is a multilevel json file which is difficult for Spark to parse using limited JVM heap space.

2. Convert the JSON file into an array of smaller individual JSON objects

   ```
   Util File for Conversion: AutoSpark/utils/json_reader.js
   ```

3. Run the spark program to get the SQLContext Schema for tweets

  ```
  Program: AutoSpark/Spark_Jobs/tweet_scanner.py
  ```
